## 🏗️ 架构亮点

### 1. 单一数据源
所有音阶和和弦定义存储在 `library.json`，JavaScript 和 Android 共享，确保一致性。

### 2. 播放中热切换
用户无需停止播放即可切换音阶或根音，显著提升用户体验。

### 3. 模块化设计
- `PlaybackSnapshotManager`: 快照管理
- `HomePlaybackStateComputer`: 状态计算
- 清晰的职责分离，易于维护

### 4. 自动化 CI/CD
GitHub Actions 自动运行测试和构建，确保代码质量。
