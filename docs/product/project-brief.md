# 习乐（Xiyue）项目简介

## 产品定位

习乐（Xiyue）是一个音乐理论练习工具，帮助用户快速选择音阶、和弦、琶音并通过键盘可视化进行练习。

## 产品形态

- 正式产品：Android App
- 辅助版本：HTML sandbox（仅内部自测，不部署）

## MVP 范围

- 12 根音
- 7 个核心音阶
- 8 个基础和弦
- 音阶上行播放
- 和弦齐奏
- 和弦琶音上行
- 基础序列播放
- BPM / 音量
- 键盘高亮
- 搜索 / 添加 / 删除 / 清空

## 当前开发策略

1. 先定义共享音乐模型与规则
2. 用可执行测试验证音乐逻辑
3. 用 HTML sandbox 快速验证流程
4. 再推进 Android 正式实现
