# 和弦规则（MVP）

以半音间隔定义：

- MajorTriad: `0, 4, 7`
- MinorTriad: `0, 3, 7`
- DiminishedTriad: `0, 3, 6`
- AugmentedTriad: `0, 4, 8`
- Maj7: `0, 4, 7, 11`
- Min7: `0, 3, 7, 10`
- Dom7: `0, 4, 7, 10`
- Min7b5: `0, 3, 6, 10`

## MVP 琶音规则

和弦琶音上行默认在高八度补一个根音：

- `C4 MajorTriad -> C4 E4 G4 C5`
- `C4 Maj7 -> C4 E4 G4 B4 C5`
