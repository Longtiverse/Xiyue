# 音阶规则（MVP）

以半音间隔定义：

- Major: `0, 2, 4, 5, 7, 9, 11, 12`
- NaturalMinor: `0, 2, 3, 5, 7, 8, 10, 12`
- HarmonicMinor: `0, 2, 3, 5, 7, 8, 11, 12`
- MelodicMinor: `0, 2, 3, 5, 7, 9, 11, 12`
- PentatonicMajor: `0, 2, 4, 7, 9, 12`
- PentatonicMinor: `0, 3, 5, 7, 10, 12`
- Blues: `0, 3, 5, 6, 7, 10, 12`
